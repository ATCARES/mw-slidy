mw-slidy provides a system for creating a HTML-slidy slideshows (http://www.w3.org/Talks/Tools/Slidy2/Overview.html#%281%29 ) as a MediaWiki extension.

Usage:

In the beginning of the slideshow add the text

  <slidy/>

this will add a right-aligned label that when pressed shows the page as a html-slideshow. 

Each level 2 mediawiki title is converted into a new slide.
